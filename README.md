# TCC (New Version) - Travel Clothing Club Landing Page

A luxury landing page redesign for Travel Clothing Club featuring:

## Design Features
- **Luxury Color Palette**: Soft cloud white, warm bronze accents, and elegant typography
- **Editorial Layout**: Magazine-style design with generous whitespace
- **Email Capture**: Prominent VIP list signup forms
- **Trust Signals**: Press mentions, testimonials, and security badges
- **Responsive Design**: Mobile-friendly layout

## Sections
1. Hero with email signup
2. Trust bar with press logos
3. How It Works (3-step process)
4. Collections (Men's, Women's, Kids')
5. Experience highlights
6. Member testimonials
7. Secondary email capture
8. Footer with navigation

## Technologies
- Pure HTML5
- CSS3 with custom properties
- Responsive grid layouts

## Setup
Simply open `index.html` in a web browser to view the landing page.

## Customization
- Update colors in CSS `:root` variables
- Replace placeholder images with actual photography
- Integrate Mailchimp form (code provided in original design)
- Add real press logos and testimonials
